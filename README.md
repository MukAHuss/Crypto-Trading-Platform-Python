## Graphical User Interface (GUI):
The GUI requires background images to run smoothly, however, Tkinter requires full file paths to implement images for the backdrop, replace the lines by pressing “Shift + F3” (#Add alternative options) and Type in the window “C:” and replace the file-paths to where you store the GUI files.

## Running the Program:
PyCharm/Visual Studio Code/IDLE: Simple Locate the “Run Python File” Command (Usually on the top right.).
Command Line:
Change the Directory to where you store the main scripts by entering ‘cd “C: File-path\File_name.exe” ‘(Example File Path)
In order to run the program smoothly, you need to run the server first, Then Client/GUI. Create an Account first, then login and test the available options.  As a measure for any errors encountered: If GUI Fails to run into any errors not accounted for, Run the Barebones GUI, If Bare-bone GUI does not run properly, Run the CLI (Command Line Interface) Version through Either IDLE’s OR CMD
